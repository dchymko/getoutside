This plugin brings a Progress Bar Block to the new WordPress Gutenberg editor.

== Description ==

Easily add a Progress Bar Block to your WordPress post or page via the new Gutenberg Editor.

- Add Progress Bar Block text
- Add Progress Bar Block %
- Add Progress Bar Block BG Color
- Progress Bar Block Text Color


== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Progress Bar Block"
3. **Install** the "Progress Bar Block" plugin
4. **Activate** "Progress Bar Block" from your Plugins page
5. **Insert Block** on your Gutenberg Editor and select "Progress Bar Block"
